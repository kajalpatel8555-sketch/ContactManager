# ContactManager

## Project Description
This is a Java Servlet and JSP based Contact Manager web application with MySQL database integration.

## Features
- Add New Contact
- View Contact List
- Form Validation
- Store Contacts in MySQL Database
- Responsive User Interface

## Technologies Used
- Java
- JSP
- Servlet
- Maven
- MySQL
- Apache Tomcat
- HTML
- CSS

## Database
Database Name: `contactdb`

Table Name: `contacts`

## How to Run
1. Import the project into IntelliJ IDEA.
2. Configure Apache Tomcat.
3. Create the MySQL database and `contacts` table.
4. Update database credentials in `DBConnection.java`.
5. Run the project on Tomcat.
